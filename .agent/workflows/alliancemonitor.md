# Alliance Monitor Workflow

This workflow provides a centralized dashboard for all alliance monitoring operations.

## Usage

Run the `/alliancemonitor` slash command to open the alliance monitoring dashboard.

## Features

The dashboard provides buttons for:

1. **Set Alliance Log Channel** - Configure where alliance change notifications are sent
2. **Select Alliance to Monitor** - Choose which alliance to track for changes
3. **View Monitoring Status** - See current monitoring configuration and stats
4. **Stop Alliance Monitoring** - Disable monitoring for a specific alliance

## Related Commands

- `/setalliancelogchannel` - Set the channel for alliance change logs
- `/selectalliance` - Select an alliance to monitor for changes
- `/alliancemonitoringstatus` - View current alliance monitoring status
- `/stopalliancemonitoring` - Stop monitoring an alliance
